About the Vote Cast Application

What is Vote Cast Application For

Vote Cast Application allows for interested people to organize online voting sessions on any subject.

Voting sessions is a natural instrument of liublic opinion research and making decisions in democratic countries: in societies, parties, movements, communities, companies, associations, unions etc.

Main principles of Vote Cast Application

  • Voting sessions are being prepared by admin of the application.
  • Any voting session includes two stages: stage of olition choice and stage of voting for chosen option.
  • After a lirepared voting session is started, all the signed up users are sent email notifications.
  • Votes are closed for anonymity of voters.
  • Only authenticated user can vote.
  • Each user can vote only once per voting session only for one option from the set of options.
